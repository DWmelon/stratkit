# iFinD 接口登录与设备说明

## SDK 接口（本项目使用的 Python/iFinDPy）

- **多设备同时登录**：不支持，**同一账号仅支持一台设备同时登录**提取数据。
- **设备绑定**：登录累计可绑定 **8 台设备**，超过会报错 **-9（被锁）**，需**联系客服解锁**。
- **登录错误码**：
  - `0`：登录成功
  - `-2`：用户名或密码错误
  - `-9`：设备数超限，账号被锁，联系客服解锁
  - `-201`：重复登录（当前账号已在其他设备登录）

## HTTP 接口

- **多设备/IP**：支持，可绑定 **20 个 IP**。
- **超限处理**：超过 IP 绑定数量时，可使用 **update access token** 接口强制更新后，再重新绑定。

## 使用建议

- 使用 SDK 时避免在多台电脑/多环境同时登录同一账号；若报 -9，先联系客服解锁，再减少已绑定设备或改用 HTTP 接口（若需多 IP）。
